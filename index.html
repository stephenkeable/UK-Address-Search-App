UK Address Search App

Mac OS X app built with Swift, using PostCoder Web API for address lookup

Simple address lookup app for Mac OS X

Allows you to search the Royal Mail address database using the PostCoder API from Allies Computing.

  1. Fork or download the project
  2. Open in Xcode,
  3. Edit the let postcoderApiKey = "PCW45-12345-12345-1234X" line to add your API key for PostCoder Web
  4. Compile or Run
  5. Search using address fragment or postcodes
  6. Choose an address from the list and see the address formatted ready for an address label

Great for small retailers or people who send out small numbers of mail or parcels and want to ensure the address is correct and formatted well for Royal Mail.

Searches use credits from your Allies account, which can be topped up on their website.

See the GitHub Page for current TO DO list etc.

If you have a website or app you can integrate PostCoder web directly into it to save your customers time when entering their address.
